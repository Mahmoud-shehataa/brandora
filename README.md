# Brandora bilingual landing page

A responsive Arabic and English landing page for Brandora marketing agency.

## Included

- `index.html` - page structure
- `styles.css` - responsive styling and RTL-aware layout
- `app.js` - language switcher, content rendering, mobile menu, and reveal effects
- `assets/brandora-logo.png` - cleaned Brandora logo asset

## Features

- English and Arabic content toggle
- Full RTL / LTR support by switching the document `dir` attribute
- Responsive hero, services, about, advantage, process, and contact sections
- No external dependencies

## How to use

Open `index.html` directly in a browser, or serve the folder with any static server.
